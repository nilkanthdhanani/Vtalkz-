User Common Features

Groups:

  • Manage groups: Provides tools to create, update, and delete groups for organizing contacts.

Whatsapp Instance:

  • Manage added WhatsApp instances: Admin can add, update, and manage WhatsApp accounts associated with the platform.
  • Connect / Disconnect WhatsApp account: Allows connecting or disconnecting a WhatsApp account from the platform.

Report Details:

1. Chart View:

  • View date-wise text, audio, video, template, group, file message report: Presents a visual chart representation of message types over time.

2. User Message:

  • Per day messages: Provides a count of messages sent per day.
  • Filter by date: Allows filtering messages based on specific dates.
  • Find user: Enables searching for messages sent by a particular user.

Campaigns History:

  • Message Content: Displays the content of messages sent as part of campaigns.
  • Sent Messages: Shows the number of messages successfully sent.
  • Pending Messages: Indicates messages waiting to be sent.
  • Total Messages: Provides a total count of all messages.
  • Seen Messages: Displays messages that have been viewed by recipients.
  • Delivered Messages: Shows messages successfully delivered to recipients.
  • Failed Messages: Indicates messages that failed to be delivered.
  • Resume Campaign: Allows resuming a paused campaign.
  • Stop Campaign: Stops an ongoing campaign.

Integration:

  • View API Docs: Provides access to API documentation for integration purposes.
  • View / Download All Videos for all features: Offers video tutorials for understanding and integrating platform features.

Profile:

  • Updates user profiles.
  • Views transferred credits.
  • Directly messages users on WhatsApp.

Whatsapp Official Services

Dashboard View:

  • Showing Remaining days: Displays the remaining days of service subscription.
  • Analysis:
    • Groups: Provides insights and analysis related to groups within the platform.
    • Contacts: Analyzes contact information and interactions.
    • Templates: Offers analysis of templates created and used.
    • Instance: Provides data and insights related to WhatsApp instances.
  • Statistics:
    • Pending message: Provides statistics related to pending messages to be send within the platform.
    • Message sent: Displays the count of total sent messages.
    • Delivered: Displays the count of total delivered messages.
    • Read message: Displays the count of total read messages.
    • Failed message: Displays the count of total failed messages.
    • Total message: Displays the count of total Total messages to send.

Templates:

  • Your Templates: Manages user-added templates, including options for editing buttons, website links, coupon codes, and contact details.
  • VTalkz Default Templates (Categorized): Manages default templates categorized for direct sharing and previewing.
  • Interactive Templates: Manages interactive templates such as product lists, catalogues, and buttons.

Shared Inbox:

  • Assign Chats to Team Members: Facilitates assigning chat responsibilities among team members.
  • Chatting with Customers: Enables communication with customers through various message types like text, media, audio, and files.
    • Message Replies: Handles replies to customer messages.
    • Message Reaction: Manages reactions to messages.
    • Medias / Text Messages / Audio Message / Files: Supports various message formats.
  • Show shared templates: Displays templates shared within the shared inbox.

Vtalkz Service

Dashboard View:

  • Showing Remaining days: Displays the remaining days of service subscription.
  • Analysis:
    • Groups: Provides insights and analysis related to groups within the platform.
    • Contacts: Analyzes contact information and interactions.
    • Templates: Offers analysis of templates created and used.
    • Instance: Provides data and insights related to WhatsApp instances.
  • Statistics:
    • Instances: Provides statistics related to instances within the platform.
    • Active Instances: Displays the count of active instances.
    • Inactive Instances: Indicates the number of inactive instances.
    • No. of Reseller: Shows the total number of resellers associated with the platform.

Templates:

  • Your Templates: Manages user-added templates, including options for editing and previewing.
  • Template Preview: Allows previewing templates before finalizing.

Message Responders:

  • Auto Responder: Manages automated responses to client messages, enhancing user interaction.
