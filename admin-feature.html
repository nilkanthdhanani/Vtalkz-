VTalkz Service

Dashboard View:

  • Showing Remaining credits / Remaining days: Displays the remaining credits and days of service subscription.

Statistics:

  • Instances: Provides statistics related to instances within the platform.
  • Active Instances: Displays the count of active instances.
  • Inactive Instances: Indicates the number of inactive instances.
  • No. of Reseller: Shows the total number of resellers associated with the platform.

Analysis:

  • Groups: Provides analysis and insights related to groups on the platform.
  • Contacts: Analyzes contact information and interactions.
  • Templates: Offers analysis of templates created and used.
  • Sent Messages: Provides statistics on messages sent.
  • Users Sent Messages: Displays statistics on messages sent by users.

Users:

  • Manage Resellers: Allows adding, updating, and viewing API keys for resellers.
    Also enables logging into the dashboard for inspection.
  • Transfer Credit: Facilitates sharing credits and provides details on credit usage.

Templates:

  • Your Templates: Manages user-added templates, including options for editing and previewing.
  • Template Preview: Allows previewing templates before finalizing.
  • VTalkz Default Templates (Categorized): Manages default templates categorized for direct sharing and previewing.

WhatsApp Instance:

  • Manage added WhatsApp instances: Admin can add, update, and manage WhatsApp accounts associated with the platform.
  • Connect / Disconnect WhatsApp account: Allows connecting or disconnecting a WhatsApp account from the platform.

Message Responders:

  • Auto Responder: Manages automated responses to client messages, enhancing user interaction.

Whatsapp Services

Dashboard View:

  • Showing Remaining days: Displays the remaining days of service subscription.

Analysis:

  • Groups: Provides insights and analysis related to groups within the platform.
  • Contacts: Analyzes contact information and interactions.
  • Templates: Offers analysis of templates created and used.
  • Instance: Provides data and insights related to WhatsApp instances.

Statistics:

  • Pending message: Shows the number of messages awaiting delivery.
  • Message sent: Displays the count of messages successfully sent.
  • Delivered: Indicates messages successfully delivered to recipients.
  • Read message: Shows messages that have been read by recipients.
  • Failed message: Indicates messages that failed to be delivered.
  • Total message: Provides a total count of all messages.

User Management:

  • Manage Roles: Administers user roles within the system.
  • Manage Permissions: Controls user access and permissions for system functionalities.
  • Manage added team members: Handles the addition and management of team members.
  • Assign Permissions to team members: Assigns specific permissions to team members based on roles.

Resellers:

  • Add / Update / Activate / Deactivate / Delete: Manages reseller accounts and their status within the system.
  • View API Key: Provides visibility into the API keys associated with resellers.
  • Login to user dashboard: Allows logging into the user dashboard from the admin panel.

Templates:

  • Your Templates: Manages user-added templates, including options for editing buttons, website links, coupon codes, and contact details.
  • VTalkz Default Templates (Categorized): Manages default templates categorized for direct sharing and previewing.
  • Interactive Templates: Manages interactive templates such as product lists, catalogues, and buttons.

Shared Inbox:

  • Assign Chats to Team Members: Facilitates assigning chat responsibilities among team members.
  • Chatting with Customers: Enables communication with customers through various message types like text, media, audio, and files.
    • Message Replies: Handles replies to customer messages.
    • Message Reaction: Manages reactions to messages.
    • Medias / Text Messages / Audio Message / Files: Supports various message formats.
  • Show shared templates: Displays templates shared within the shared inbox.

Common Features in Both Services

Whatsapp Instance:

  • Manage added WhatsApp instances: Admin can add, update, and manage WhatsApp accounts associated with the platform.
  • Connect / Disconnect WhatsApp account: Allows connecting or disconnecting a WhatsApp account from the platform.

Groups:

  • Manage groups: Provides tools to create, update, and delete groups for organizing contacts.

1. Add contacts:

  • Individual: Adds contacts one by one to specific groups.
  • Bulk Upload (.xlsx): Enables uploading contacts in bulk using an Excel file.
  • View group contacts: Displays the contacts within each group.
  • Export contacts: Allows exporting group contacts to a file.
  • Download contact upload sample file: Provides a sample Excel file for bulk contact uploads.

Send Message:

1. Compose Message:

  • Send Individual Message: Sends a message to a single contact or number.
  • Submit country code and contact: Requires entering the country code and contact number for messaging.
  • Schedule / Quick Send: Offers options to schedule messages for later sending or send them immediately.

2. Send Bulk Message (by already created groups):

  • Select group for bulk messaging: Selects a group to send a message to multiple contacts at once.
  • Schedule / Quick Send: Provides scheduling or immediate sending options for bulk messages.

Scheduled Messages:

  • View scheduled messages: Displays messages scheduled for future sending.

Report Details:

1. Chart View:

  • View date-wise text, audio, video, template, group, file message report: Presents a visual chart representation of message types over time.

2. User Message:

  • View user-wise messages: Displays messages sent by specific users.
  • Per day messages: Provides a count of messages sent per day.
  • Filter by date: Allows filtering messages based on specific dates.
  • Find user: Enables searching for messages sent by a particular user.

Campaigns History:

  • Message Content: Displays the content of messages sent as part of campaigns.
  • Sent Messages: Shows the number of messages successfully sent.
  • Pending Messages: Indicates messages waiting to be sent.
  • Total Messages: Provides a total count of all messages.
  • Seen Messages: Displays messages that have been viewed by recipients.
  • Delivered Messages: Shows messages successfully delivered to recipients.
  • Failed Messages: Indicates messages that failed to be delivered.
  • Resume Campaign: Allows resuming a paused campaign.
  • Stop Campaign: Stops an ongoing campaign.

Integration:

  • View API Docs: Provides access to API documentation for integration purposes.
  • View / Download All Videos for all features: Offers video tutorials for understanding and integrating platform features.

Payment History:

  • View payment history for purchased plans: Shows the history of payments made for subscribed plans.
  • Generate Invoice and view purchase details: Enables generating invoices and viewing detailed purchase information.
