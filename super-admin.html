Super Admin Features

Manage All Users:

  • Lists all registered users with their details.
  • Super Admin can add new users with the same registration details.
  • API keys for all users are available and listed.
  • Super Admin can log in to any user's dashboard for inspection and return to the Super Admin dashboard.

Manage Users and Their Resellers:

  • Enables the transfer of credits between users and resellers.
  • Allows viewing user credits and transferring them as requested.

Manage All Instances (WhatsApp accounts):

  • Provides details of all WhatsApp instances.
  • Allows updating and deleting instances as needed.

Manage Templates:

  • Adds, updates, deletes, and previews templates.
  • Manages categories and subcategories for templates.

View Reports:

1. Chart View:

  • Displays date-wise reports of text, audio, video, template, group, and file messages.

2. User Message:

  • Views user-wise messages, including per-day messages.
  • Filters messages by date and finds specific users.

Integration:

  • Manages API documentation and feature videos for integration purposes.

Emergency Contact:

  • Sends messages to all users via WhatsApp in case of emergency.

Website Content:

  • Manages news, blogs, inquiries, new features, and website homepage content.

Plans:

  • Manages all plans purchased by users.

Payment History:

  • Views the payment history for purchased plans.
  • Generates invoices and views purchase details.

Profile:

  • Updates user profiles.
  • Views transferred credits.
  • Directly messages users on WhatsApp.
